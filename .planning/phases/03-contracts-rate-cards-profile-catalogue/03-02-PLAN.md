    ```python
    class RateCeilingExceeded(Exception):
        """Raised when a candidate's rate exceeds the active rate card ceiling."""
        def __init__(self, candidate_rate: Decimal, ceiling: Decimal,
                     profile_code: str, sfia_level: int):
            self.candidate_rate = candidate_rate
            self.ceiling = ceiling
            self.profile_code = profile_code
            self.sfia_level = sfia_level
    ```

    `async def check_rate_ceiling(db, contract_id, profile_id, sfia_level, candidate_rate, today=None) -> None`:
    - If contract_id is None -> return immediately (no constraint)
    - If profile_id is None -> return immediately
    - If candidate_rate is None -> return immediately (see RESEARCH.md Pitfall 3)
    - Query active rate card entry: WHERE contract_id = ? AND profile_id = ? AND sfia_level = ? AND effective_from <= today AND effective_to >= today. Use `.limit(1)`.
    - If no entry found -> return (no constraint applies)
    - If candidate_rate > entry.max_daily_rate -> raise RateCeilingExceeded with all context

    `async def has_approved_rate_override(db, demand_id, candidate_id) -> bool`:
    - Query ApprovalRequest WHERE type='rate_override' AND status='approved' AND context_data contains matching demand_id and candidate_id.
    - For JSONB containment: use `ApprovalRequest.context_data["demand_id"].as_string() == str(demand_id)` or cast to text. The simplest approach for SQLite test compat: load all rate_override approved approvals and filter in Python. For production, use JSONB operators. Since tests use SQLite (no JSONB operators), use the Python-filter approach:
      ```python
      stmt = select(ApprovalRequest).where(
          ApprovalRequest.type == "rate_override",
          ApprovalRequest.status == "approved",
      )
      result = await db.execute(stmt)
      for approval in result.scalars().all():
          ctx = approval.context_data or {}
          if ctx.get("demand_id") == str(demand_id) and ctx.get("candidate_id") == str(candidate_id):
              return True
      return False
      ```
