    **Helper function — compute profile diff:**
    ```python
    def compute_profile_diff(snapshot: dict, demand_values: dict) -> list[dict]:
        """Compare profile snapshot against demand field values.
        Returns list of {field, profile_value, demand_value} for deviations."""
        # Map snapshot fields to demand field equivalents
        field_mappings = {
            "sfia_level_min": "sfia_level_min",
            "sfia_level_max": "sfia_level_max",
            "min_years_exp": "min_years_exp",
            "min_education": "min_education",
            "required_clearance": "required_clearance",
            "description": "description",
        }
        diffs = []
        for snapshot_field, demand_field in field_mappings.items():
            profile_val = snapshot.get(snapshot_field)
            demand_val = demand_values.get(demand_field)
            if profile_val != demand_val:
                diffs.append({
                    "field": snapshot_field,
                    "profile_value": profile_val,
                    "demand_value": demand_val,
                })
        return diffs
    ```

    Implement the advisory compliance checker as described in RESEARCH.md Pattern 5. This is a rule-based checker (NOT AI-powered) that compares profile requirements against candidate parsed data.

    ```python
    from typing import Any
    from app.models.profile_requirement import ProfileRequirement

    def check_profile_compliance(
        requirements: list[ProfileRequirement],
        candidate_data: dict[str, Any],
    ) -> dict:
        """Run advisory compliance check of candidate data against profile requirements.
        
        Returns:
            {
                "overall": "MET" | "PARTIALLY_MET" | "NOT_MET",
                "items": [
                    {
                        "req_id": str(UUID),
                        "req_type": str,
                        "description": str,
                        "is_mandatory": bool,
                        "status": "MET" | "PARTIALLY_MET" | "NOT_MET"
                    }
                ]
            }
        
        Advisory only — never raises exceptions or blocks operations.
        Called by Phase 6 shortlist compliance endpoint.
        """
    ```

    **Evaluation rules per req_type:**
    
    - **skill**: Check if candidate_data["skills"] (string or list) contains the requirement description text (case-insensitive substring match). MET if found, NOT_MET if not found.
    - **certification**: Check if candidate_data["certifications"] (list of strings) contains the description (case-insensitive). MET if exact match, PARTIALLY_MET if partial match, NOT_MET if absent.
    - **language**: Check if candidate_data["languages"] (list of dicts with "language" and "cefr_level") has an entry matching the description language. If found, compare CEFR level: MET if candidate level >= required min_cefr_level (A1 < A2 < B1 < B2 < C1 < C2), PARTIALLY_MET if candidate has the language but lower level, NOT_MET if language not found.
    - **clearance**: Check if candidate_data["clearances"] (list of dicts or strings) mentions the required clearance. MET if found with matching or higher level, NOT_MET if not found.
    - **education**: Check if candidate_data["education"] mentions the required education level. Simple substring match. MET if found, NOT_MET if not found.

    **CEFR level comparison helper:**
    ```python
    _CEFR_ORDER = {"A1": 1, "A2": 2, "B1": 3, "B2": 4, "C1": 5, "C2": 6}
    
    def _cefr_meets_minimum(candidate_level: str, required_level: str) -> bool:
        return _CEFR_ORDER.get(candidate_level.upper(), 0) >= _CEFR_ORDER.get(required_level.upper(), 0)
    ```

    **Overall status logic:**
    - If ANY mandatory requirement is NOT_MET → overall = NOT_MET
    - If ANY requirement is PARTIALLY_MET → overall = PARTIALLY_MET
    - Otherwise → overall = MET

    **Handle missing candidate_data gracefully:** If candidate_data is empty dict or None, all requirements are NOT_MET. This function must NEVER raise an exception — it is advisory.
