      - For each matching clearance:
        - Log the alert: `logger.info(f"Clearance alert ({days}d): candidate={clearance.candidate_id}, clearance={clearance.id}, expiry={clearance.expiry_date}")`

      - Commit after each threshold batch
    - Wrap entire function in try/except with logger.error on failure
    - Do NOT re-raise — scheduler jobs should not crash the scheduler

      # After cv_file is persisted:

          await parse_cv_file(cv_file.id, candidate.id, db)
