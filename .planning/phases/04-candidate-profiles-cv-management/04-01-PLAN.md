    Create 4 new ORM model files following the established pattern in models/contract.py (SQLAlchemy 2.0 Mapped syntax, UUID primary keys, created_at/updated_at timestamps).

    CRITICAL: Candidates are GLOBAL. These tables do NOT have a tenant_id column. This is intentional and differs from all Phase 3 models.

    **backend/app/models/candidate.py** — `class Candidate(Base)`:
    - `__tablename__ = "candidates"`
    - `id`: UUID PK (default uuid4)
    - `user_id`: UUID FK to users.id (UNIQUE, nullable=True, ondelete="SET NULL") — one user = one candidate profile
    - `full_name`: String(255) NOT NULL
    - `email`: String(255) nullable (may differ from user.email)
    - `phone`: String(50) nullable
    - `location`: String(255) nullable
    - `availability`: String(100) nullable (e.g., "immediately", "2 weeks", ISO date)
    - `rate_expectation`: Numeric(10, 2) nullable
    - `rate_currency`: String(3) nullable, default "EUR"
    - `sfia_level`: Integer nullable (1-7, confirmed by candidate)
    - `parsed_data`: JSONB nullable — full AI-parsed CV data (entire ParsedCV structure)
    - `parsed_skills_esco`: JSONB nullable — ESCO codes from AI parsing, NEVER returned in API responses
    - `profile_complete`: Integer nullable, default 0 — weighted completeness percentage (0-100)
    - `visibility`: JSONB nullable — `{"industries": [...], "tenant_types": [...]}` controlling which tenants can discover this candidate
    - `created_at`, `updated_at` — DateTime(timezone=True) with server_default=func.now()
    - NO tenant_id. NO RLS. This is a global entity.

    **backend/app/models/cv_file.py** — `class CvFile(Base)`:
    - `__tablename__ = "cv_files"`
    - `id`: UUID PK
    - `candidate_id`: UUID FK to candidates.id (NOT NULL, ondelete="CASCADE")
    - `version`: Integer NOT NULL
    - `is_current`: Boolean NOT NULL, default=False
    - `original_filename`: String(500) NOT NULL
    - `content_type`: String(100) NOT NULL (e.g., "application/pdf")
    - `file_size`: Integer NOT NULL (bytes)
    - `blob_path`: String(1000) NOT NULL — Azure Blob Storage path (e.g., "cvs/{candidate_id}/v{N}_original.pdf")
    - `raw_text`: Text nullable — extracted text from CV (for re-parsing without re-downloading)
    - `parse_status`: String(20) NOT NULL, default="pending" — values: pending, parsing, completed, failed
    - `parse_error`: Text nullable — error message if parsing failed
    - `created_at`, `updated_at`
    - Add `__table_args__ = (UniqueConstraint("candidate_id", "version", name="uq_cv_candidate_version"),)` to prevent concurrent upload version conflicts

    **backend/app/models/security_clearance.py** — `class SecurityClearance(Base)`:
    - `__tablename__ = "security_clearances"`
    - `id`: UUID PK
    - `candidate_id`: UUID FK to candidates.id (NOT NULL, ondelete="CASCADE")
    - `level`: String(50) NOT NULL — e.g., "SECRET", "TOP SECRET", "NATO SECRET"
    - `clearance_type`: String(50) NOT NULL — e.g., "NATO", "EU", "National"
    - `issuing_authority`: String(100) NOT NULL
    - `reference_number`: String(100) nullable
    - `issue_date`: Date nullable
    - `expiry_date`: Date nullable
    - `status`: String(30) NOT NULL, default="self_declared" — values: self_declared, pending_verification, verified, expired, revoked
    - `application_status`: String(30) nullable — for clearances currently being applied for
    - `alert_sent_90d`: Boolean NOT NULL, default=False
    - `alert_sent_60d`: Boolean NOT NULL, default=False
    - `alert_sent_30d`: Boolean NOT NULL, default=False
    - `created_at`, `updated_at`

    **backend/app/models/candidate_language.py** — `class CandidateLanguage(Base)`:
    - `__tablename__ = "candidate_languages"`
    - `id`: UUID PK
    - `candidate_id`: UUID FK to candidates.id (NOT NULL, ondelete="CASCADE")
    - `language`: String(100) NOT NULL — e.g., "English", "French"
    - `cefr_level`: String(2) NOT NULL — A1, A2, B1, B2, C1, C2
    - `source`: String(20) NOT NULL, default="self_declared" — values: self_declared, test, ai_inferred
    - `test_name`: String(255) nullable — e.g., "IELTS", "DELF"
    - `test_score`: String(50) nullable
    - `test_date`: Date nullable
    - `created_at`, `updated_at`
